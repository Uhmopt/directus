# Avatar

```html
<v-avatar>RVZ</v-avatar>

<v-avatar>
	<img src="..." />
</v-avatar>

<v-avatar>
	<v-icon name="person">
</v-avatar>
```

## Sizes / Colors

The avatar component supports multiple sizes and colors. The color prop accepts any valid CSS color. CSS variable names can be passed as well.

| Prop Name      | Size in PX |
|----------------|------------|
| `x-small`      | 32         |
| `small`        | 40         |
| None (default) | 48         |
| `large`        | 56         |
| `x-large`      | 64         |
